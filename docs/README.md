# Hanzo App Documentation

Welcome to the official documentation for **Hanzo App** - the AI Command Menu that works with your OS.

## Overview

Hanzo App combines leading AI models with powerful extensions right on your operating system. It provides:

- **AI Chat** - Access to 32+ leading models in one interface
- **Quick AI** - Instant AI access with a single hotkey
- **AI Extensions** - Turn natural language into actions
- **AI Commands** - Automate tasks with custom commands

## Product Ecosystem

- **Hanzo App** - The desktop application (this project)
- **Hanzo.app** - Our website and web interface
- **Hanzo.ai** - The AI cloud platform and SaaS backend

## Architecture

Hanzo App is built with:
- **Tauri** - Native performance and security
- **React** - Modern UI framework
- **MCP** - Model Context Protocol for AI tools
- **LLM Router** - Unified access to 100+ providers
