## Highlights 🎉

- We have added Model Context Protocol (MCP) support to the stable build of Hanzo AI. It needs to be enabled in the General Settings tab.
- Hanzo AI now supports hanzoai's latest model, Hanzo AI-Nano-128k.
- Some hot fixes and improvements.

Update your Hanzo AI or [download the latest](https://hanzo.ai/).

For more details, see the [GitHub release notes](https://github.com/hanzoai/hanzo/releases/tag/v0.6.3).
