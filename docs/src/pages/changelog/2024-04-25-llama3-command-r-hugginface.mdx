Hanzo AI has added compatibility with Meta’s open-source language model, `Llama3`, through the integration with `llamacpp` (thanks to [@ggerganov](https://github.com/ggerganov)). 

Additionally, `Command R+` is now supported. It is the first open-source model to surpass GPT-4 on the [LMSys leaderboard](https://chat.lmsys.org/?leaderboard).

![Commandr](/assets/images/changelog/hanzo_cohere_commandr.gif)

## Import Huggingface models directly

Users can now import Huggingface models into Hanzo AI. Simply copy the model’s link from Huggingface and paste it into the search bar on Hanzo AI Hub.

![HugginFace](/assets/images/changelog/hanzo_hugging_face.gif)

## Enhanced LaTeX understanding

Hanzo AI now understands LaTeX, allowing users to process and understand complex mathematical expressions more effectively. 

![Latex](/assets/images/changelog/hanzo_update_latex.gif)

For more details, see the [GitHub release notes](https://github.com/hanzoai/hanzo/releases/tag/v0.4.12).
