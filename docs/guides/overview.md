# Hanzo App Overview

## AI Command Menu - The Future of Computer Control

Hanzo App is the AI Command Menu that works with your OS, bringing frontier AI models and intelligent automation to your fingertips. We make frontier AI models and tools accessible through a beautiful, native interface.

## Key Features

### 🎯 AI Command Menu (⌘+Space)
Your intelligent command center powered by AI:
- **Smart App Launcher**: AI understands natural language "open my work apps"
- **Intelligent File Search**: "find that PDF from last week"
- **AI-Powered Actions**: Execute complex commands with simple language
- **Smart Calculator**: Natural language math "15% tip on $80"
- **AI Window Management**: "organize my desktop for coding"

### 🤖 AI Chat - 32+ Models in One Interface
Your new virtual assistant with:
- **32+ Leading Models**: GPT-4, Claude 4, Gemini, and more
- **Fine-tuned Presets**: Optimized for different tasks
- **Model Comparison**: Compare responses side-by-side
- **PDF, CSV & Screen Attachments**: Work with any content
- **Local-First Privacy**: Your data stays on your device

### 🔧 MCP Integration
Model Context Protocol server that:
- **Exposes 50+ Tools**: File operations, app control, system management
- **Claude Desktop Compatible**: Use with Anthropic's Claude
- **Extensible Framework**: Add custom tools easily
- **Secure by Design**: Sandboxed operations with permission controls

### 🌐 LLM Router
Unified AI model access:
- **100+ Providers**: OpenAI, Anthropic, Google, AWS, and more
- **Smart Routing**: Automatic failover and load balancing
- **Local Models**: Privacy-first with llama.cpp
- **Cost Optimization**: Choose the best model for your needs

### ⚡ Quick AI - Instant AI Access
Ask anything, anytime, anywhere:
- **Single Hotkey Access**: ⌘+Option+Space for floating AI
- **Always Above Apps**: AI assistant that follows you
- **Web Search Integration**: Real-time information
- **Instant Responses**: Optimized for speed

### 🔧 AI Extensions
Turn natural language into actions:
- **Built-in Extensions**: Control your OS naturally
- **Extension Store**: Community-powered capabilities
- **Custom Development**: Build your own AI tools
- **Multi-Extension Workflows**: Chain actions together

### 🎯 AI Commands
Your automation assistant:
- **30+ Built-in Commands**: Common tasks automated
- **Custom Commands**: Create your own workflows

## Getting Started

1. **AI Command Menu**: Press ⌘+Space
2. **AI Chat**: Press ⌘+Shift+Space  
3. **Quick AI**: Press ⌘+Option+Space
4. **Try Natural Language**:
   - "Open my coding setup"
   - "Find yesterday's documents"
   - "Minimize everything except Chrome"
   - "Calculate 20% tip on $85.50"
