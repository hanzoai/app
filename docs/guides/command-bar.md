# AI Command Menu Guide

The AI Command Menu is your intelligent command palette for controlling your computer - think of it as ⌘+K powered by AI for your entire system.

## Opening the AI Command Menu

Press **⌘+Space** to open the AI Command Menu from anywhere on your system.

## Features

### 🚀 App Launching
Simply type the name of any application to find and launch it:
- `safari` - Launch Safari
- `ter` - Finds Terminal, iTerm, etc.
- `code` - Launch VS Code

### 📁 File Search
Start your query with `/` to search files:
- `/meeting notes` - Find files with "meeting notes"
- `/pdf` - Find all PDF files
- `/today` - Files modified today

### 🪟 AI Window Management
Use the `>` prefix for intelligent window commands:
- `>minimize all` - Minimize all windows
- `>focus safari` - Bring Safari to front
- `>tile` - Arrange windows in tiles
- `>list` - Show all open windows
- AI understands natural language: "hide everything except Chrome"

### 🔧 System Commands
Quick system actions:
- `quit finder` - Quit an application
- `empty trash` - Empty the trash
- `sleep` - Put computer to sleep
- `lock` - Lock the screen
- `restart` - Restart computer

### 🧮 Calculator
Type any math expression:
- `25 * 4` → 100
- `sqrt(144)` → 12
- `15% of 200` → 30
- `$50 + 20%` → $60

### 🔍 Smart Search
The command bar learns from your usage:
- Frequently used items appear first
- Fuzzy matching for typos
- Recent items are prioritized

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| ⌘+Space | Open AI Command Menu |
| ↑/↓ | Navigate results |
| Enter | Execute selected |
| ⌘+Enter | Execute and hide |
| Tab | Autocomplete |
| Esc | Close AI Command Menu |
| ⌘+C | Copy selected item |
| ⌘+Number | Quick select (1-9) |

## Advanced Usage

### Aliases
Create shortcuts for common actions:
- `gm` → Launch "Google Meet"
- `docs` → Open Documents folder
- `dl` → Open Downloads

### Chains
Combine multiple actions:
- `quit all && empty trash` - Quit all apps and empty trash
- `lock && sleep` - Lock screen then sleep

### Parameters
Some commands accept parameters:
- `timer 5m` - Set 5-minute timer
- `remind Call John at 3pm` - Create reminder
- `note Quick idea for project` - Create note

## Tips & Tricks

1. **Quick Launch**: Type just the first few letters
2. **Smart Matching**: "vsc" matches "Visual Studio Code"
3. **History**: Use ↑ to access recent commands
4. **Pinning**: Pin frequently used items to top
5. **Direct Paths**: Type full paths like `/Users/me/Documents`

## Troubleshooting

### AI Command Menu Won't Open
1. Check keyboard shortcut conflicts in System Preferences
2. Ensure Hanzo App has accessibility permissions
3. Try alternative shortcut (configurable in settings)

### Slow Search Results
1. Rebuild search index in settings
2. Limit search scope to specific folders
3. Disable real-time file indexing if needed

### Missing Apps
1. Refresh app list in settings
2. Check Applications folder permissions
3. Manually add app paths if needed
